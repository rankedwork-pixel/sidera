# Creative Decision Framework

## The Three Questions

For every creative, answer these in order:

### 1. Do we have enough data?
- DTC: >= 1,000 impressions AND >= 7 days
- B2B: >= 5,000 impressions AND >= 14 days
- Video: >= 14 days regardless of vertical
- If NO → classify as Probation, do not cut

### 2. Is it converting at an acceptable cost?
- Compare backend CPA to target CPA (from account goals)
- Compare backend ROAS to break-even ROAS
- If below target but above break-even → Maintain
- If below break-even with sufficient data → Cut
- If above target → candidate for Scale

### 3. Is it fatiguing?
- Check frequency trend (rising above 3.0?)
- Check CTR trend (declining 5+ consecutive days?)
- Check CPM trend (rising = auction competition or audience saturation)
- If fatiguing but still profitable → reduce budget, don't kill
- If fatiguing AND below target → Cut immediately

## Common Mistakes to Avoid

1. **Cutting too early** — A creative with 500 impressions and 0
   conversions is not "failing." It has insufficient data. Wait.

2. **Ignoring the inflation ratio** — Platform says ROAS 5.0x but
   backend says 2.1x. The real number is 2.1x. Make decisions on that.

3. **Scaling too aggressively** — Never increase budget more than 20%
   at a time. Large jumps reset Meta's learning phase and can tank
   performance for 3-5 days.

4. **Comparing apples to oranges** — A prospecting creative and a
   retargeting creative have fundamentally different CPA ranges. Always
   compare within the same funnel stage.

5. **Ignoring creative concentration** — If one creative is 50%+ of
   spend, any issue with that creative = account-level crisis. Always
   flag concentration risk.

6. **Chasing CTR** — High CTR with poor conversion rate means the
   creative attracts clickers who don't buy. Backend CPA is what
   matters, not click volume.

## Recommendation Priority Order

When resources are limited, prioritize actions in this order:
1. **Cut money-losing creatives** (immediate savings)
2. **Address fatigue on profitable creatives** (prevent future losses)
3. **Scale proven winners** (increase revenue)
4. **Launch new tests** (future pipeline)

Never skip step 1 to do step 3. Stop the bleeding first.
